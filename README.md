# jumping-pygame
# Jednoduchá 2D skákačka

Jednoduchá 2D hra v Pythonu s využitím OOP a knihovny tkinter.

Hráč ovládá postavu, která přeskakuje překážky a sbírá skóre.

---

## Funkce

- 2D grafika  
- Ovládání klávesnicí  
- Skákání přes překážky  
- Detekce kolizí  
- Počítání skóre  
- Restart hry  

---

## Ovládání

- Mezerník – skok  
- R – restart
